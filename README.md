# GDG Hackathon - Medak

**Medak** is an AI-powered emergency relay for deaf and speech-impaired people in Serbia. Two Gemini 2.0 Flash Live agents coordinate to bridge the gap between a non-speaking caller and a 112 emergency dispatcher.

Built for the Google "Build with AI" Hackathon 2026, Belgrade.

## Directory Structure

```
medak/          Code repository (backend + frontend + infra)
docs/
  research/     P09 deep-dive: GDPR, B2G model, Serbian market, sensory pain points
  solutions/    Solution architectures and scoring for P09 emergency relay
  strategy/     Hackathon guide, competitive analysis, problem scoreboard, checklist
```
